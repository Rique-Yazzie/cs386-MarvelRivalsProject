NVIDIA GeForce NOW is live on Marvel Rivals now!

January 10, 2025

Dear Rivals,

Exciting news for Season 1! Marvel Rivals is now supported on NVIDIA GeForce NOW, NVIDIA's cloud gaming service. Transform nearly any device into a high-performance, GeForce-powered gaming rig.

How to Play Marvel Rivals on NVIDIA GeForce NOW:

  1. Players need to sign up for a NVIDIA GeForce NOW account. If you already have a NVIDIA GeForce NOW account, please go straight to step 3.
  2. New NVIDIA GeForce NOW users will need a NVIDIA GeForce NOW membership to experience Marvel Rivals. For more details related to NVIDIA GeForce NOW membership, please refer to the following website page for more details: https://www.nvidia.com/en-us/geforce-now/memberships/
  3. After you have your Free, Performance, or Ultimate GeForce NOW membership, you are ready to get in and experience the game. Download NVIDIA GeForce NOW app, or launch GeForce NOW in a supported browser, and Check out Marvel Rivals!
    https://www.nvidia.com/en-us/geforce-now/download/

Note: Marvel Rivals is a free-to-play game, however, users wishing to experience Marvel Rivals on the NVIDIA GeForce NOW platform will need a GeForce NOW membership. The specific terms are subject to the policies set forth by the provider of NVIDIA GeForce NOW service.

For more detailed information, please refer to the following website page for more details:
